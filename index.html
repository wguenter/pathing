Pathing Visualizations for Computer Games



Thank you for taking part in our study on visual techniques for representing the pathing decisions in 3D adventure games via Prolific!

Study Procedure

  1. Please first download the game prototype from Google Drive
  2. Extract the downloaded .zip file into a folder your choice.
  3. Start the game via Windows/SolGameDemo.exe.
  4. The game will then guide you through the study. All survey questions are directly integrated into the game. Amongst others the game will ...
    • ask you to provide your consent (main points are reproduced below)
    • enter your Prolific ID
    • basic demographics
    • questions regarding your play experience
  5. When the game is completed it will show you the Prolific Completion Code. Please make sure to take note of it!
